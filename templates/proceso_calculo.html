Distribución de Poisson

Usando la distribución de Poisson, calculamos la probabilidad de que cada equipo anote al menos 1 gol. La fórmula de Poisson es: P(X=k) = (e^-λ × λ^k) / k!

P(equipo NO marque) = P(X=0) = e^(-λ)
P(equipo SÍ marque) = 1 - P(X=0) = 1 - e^(-λ)
{{ data.team1.name }}
Probabilidad de NO marcar:
P(X=0) = e^(-{{ "%.2f"|format(data.team1.lambda|float) }}) = {{ "%.4f"|format(2.718281828459045 ** (-(data.team1.lambda|float))) }}
Probabilidad de SÍ marcar (≥1 gol):
P(X≥1) = 1 - {{ "%.4f"|format(2.718281828459045 ** (-(data.team1.lambda|float))) }} = {{ "%.4f"|format(1 - (2.718281828459045 ** (-(data.team1.lambda|float)))) }}
Probabilidad de marcar {{ "%.1f"|format(data.team1.probScores|float) }}%
{{ data.team2.name }}
Probabilidad de NO marcar:
P(X=0) = e^(-{{ "%.2f"|format(data.team2.lambda|float) }}) = {{ "%.4f"|format(2.718281828459045 ** (-(data.team2.lambda|float))) }}
Probabilidad de SÍ marcar (≥1 gol):
P(X≥1) = 1 - {{ "%.4f"|format(2.718281828459045 ** (-(data.team2.lambda|float))) }} = {{ "%.4f"|format(1 - (2.718281828459045 ** (-(data.team2.lambda|float)))) }}
Probabilidad de marcar {{ "%.1f"|format(data.team2.probScores|float) }}%

Regresión Logística

La regresión logística analiza múltiples factores simultáneamente y usa la función sigmoide para convertir una combinación lineal de características en una probabilidad entre 0% y 100%.

z = w₀ + w₁×f₁ + w₂×f₂ + ... + wₙ×fₙ
P(BTTS) = σ(z) = 1 / (1 + e^(-z))
Características normalizadas y ponderadas:
• Goles anotados ambos equipos (peso: 0.45 cada uno)
• Goles recibidos ambos equipos (peso: 0.35 cada uno)
• Tiros a puerta (peso: 0.08 cada uno)
• Promedio de goles del partido (peso: 0.25)
• Fuerza ofensiva combinada (peso: 0.15)
Cálculo de z (combinación lineal):
z = -0.8 + ({{ data.team1.stats.goalsScored }}×0.45) + ({{ data.team2.stats.goalsScored }}×0.45) + ...
z ≈ valor calculado por el modelo
Aplicar función sigmoide:
P(BTTS) = 1 / (1 + e^(-z)) = {{ "%.2f"|format(data.btts.logistic|float) }}%
